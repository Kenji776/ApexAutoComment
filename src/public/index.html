Apex Auto Comment & Documentation Generator

This utility will automatically comment your Salesforce Apex classes and generate documentation from the resulting commented code. Simply select a folder containing Apex classes, INCLUDING the metadata files, select if you want comments, documentation, or both and let it run. Be aware that the document generation depends on specifically formatted comments using the javadoc standard.
